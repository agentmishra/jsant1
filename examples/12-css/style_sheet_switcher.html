A Stylish Heading

This Style Guide for the Branch Libraries of the New York Public Library explains the markup and design requirements for all Branch Libraries web projects, along with various standards and best practices.

Branch Libraries projects must be authored in valid XHTML 1.0 Transitional and styled with valid Cascading Style Sheets. See the XHTML and CSS sections below for details. Additional sections of this Style Guide, coming soon, will provide information on writing for the web, naming and filing your documents, and other useful topics and guidelines.
